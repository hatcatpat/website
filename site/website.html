hi, this is my website. did you find the place okay? here, let me take your coat...

this site is generated by a simple c program that automagically inserts the commonly used code into every page (i.e, things like the title, menu, stylesheet, etc).

it also has some pointlessly cute (and perhaps aggravating) features involving randomness: the theme and numbering-style are randomized each time a page is generated. the site also has some style choices, such as

  • 3 colors: fg, bg, and accent
  • the first letter of each p tag is highlighted
  • code is denoted by italics and accent-color, $ indicates it is shell command
  • inverted colors for emphasis
  • all lowercase

i'm certain new eccentricities will emerge over time :^)

the program itself is just a series of fputs commands, with the occasional loading of a file. its usage is $ ./generate page [theme] where 'page' is the name of the page to generate (i.e, in this case its 'website'), and 'theme' is optional and will select the theme.

i used a bash script, build.sh to $ ./generate all the files in the pages directory. it uses this command (which i wholeheartedly knicked from stackoverflow), 

$ ls ../site/themes | sort -r | tail -$n | while read file; do
  blahblahblahblah
done
to output a randomly sorted list of the .css themes. i then take the first one and use that as the theme!

remark: pipe | and redirection > are neat
